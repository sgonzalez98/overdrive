Enable RoadSense
Detects road hazards (bumps, potholes, speed breakers) from motion and GPS, and warns you ahead of known ones. All other RoadSense settings only take effect while this is on.
Map
Open map
Open the full-screen map to search routes, navigate, and see road hazards along the way. Tap any hazard to confirm it or remove it.
Routing
No routing key
The map basemap is free. Turn-by-turn routing needs a free personal key from a Valhalla provider — paste yours here. It is optional: the map and hazards work without it. Your key is stored encrypted on-device.
How to get a free routing key
  1. Tap "Get a free key" to open the Stadia Maps signup, then create a free account (no credit card for the free tier).
  2. In the dashboard, open Properties and create a Property, then go to Authentication and API Keys.
  3. Click "Create API key" and copy the key string it shows you.
  4. Paste it into the "Routing API key" field below and tap Save. Leave the endpoint as the default.
Routing endpoint
The Valhalla-compatible routing URL. Leave as the default unless your provider gives you a different one.
Routing API key
Your personal BYOK key. Stored encrypted on-device and never shown again after saving.

Detection
Detection Sensitivity
How easily RoadSense flags a bump, pothole or speed breaker. 50% is the default tuning. Slide toward More to catch gentler features (and a few more false detections); slide toward Less to only flag clear, hard hits. This changes what gets detected — not the same as the warning Minimum Confidence below.

Calibration Mode
Confirm detections in the overlay
Shows each detected hazard on the screen overlay with confirm / correct buttons so you can teach RoadSense which detections are real. Use this for a few drives to tune detection for your vehicle, then turn it off.
On-screen overlay
Show overlay on screen
Shows the floating RoadSense pill and hazard card on the screen. Turn this off to hide it — detection, audio warnings, and crowdsourcing keep running in the background; only the on-screen display is hidden.

Crowdsource
By default RoadSense keeps everything on this device. Upload and download are separate — turning one on does not turn on the other.
Upload my detections
Sends anonymized hazard detections off-device to improve the shared map. Off by default — RoadSense is on-device-only until you enable this.
Download shared hazards
Pulls hazards reported by other drivers so you get warned about hazards you haven't driven over yet. Off by default.
Sync worker URL
Optional. Point upload/download at your own sync worker instead of the default endpoint.
Calibration Data
Delete local calibrations
Removes the hazard detections and calibrations stored on this device. Cannot be undone.
Delete cloud calibrations
Removes the detections you uploaded from the shared cloud map. Other drivers' data is unaffected. Cannot be undone.
